
| Chức năng                          | Method | URL               | Query params (nếu có)       | Status thành công | Status lỗi (ví dụ)              |
|------------------------------------|--------|-------------------|-----------------------------|------------------|---------------------------------|
| Lấy danh sách sách                 | GET    | /books            | ?page=1&limit=10&author=... | 200 OK           | 400 Bad Request (tham số sai)   |
| Thêm sách mới                      | POST   | /books            |                             | 201 Created      | 400 Bad Request (thiếu dữ liệu) |
| Lấy chi tiết một sách              | GET    | /books/{id}       |                             | 200 OK           | 404 Not Found                   |
| Cập nhật toàn bộ thông tin sách    | PUT    | /books/{id}       |                             | 200 OK           | 404 Not Found                   |
| Cập nhật số lượng sách             | PATCH  | /books/{id}       |                             | 200 OK           | 404 Not Found                   |
| Xóa sách                           | DELETE | /books/{id}       |                             | 204 No Content   | 404 Not Found                   |
| Lấy danh sách thẻ mượn của một sách| GET    | /books/{id}/loans | ?page=1&limit=10            | 200 OK           | 404 Not Found                   |
| Tạo thẻ mượn mới                   | POST   | /loans            |                             | 201 Created      | 400 Bad Request                 |
| Trả sách (cập nhật ngày trả)       | PATCH  | /loans/{id}       |                             | 200 OK           | 404 Not Found                   |
